# Python 코드 설명기

이 리포지토리는 Google Gemini API를 사용하여 사용자가 입력한 Python 코드의 역할을 설명해주는 간단한 CLI 애플리케이션입니다.

## 파일 설명

### `app.py`

- **설명:** 메인 애플리케이션 파일입니다. 사용자로부터 Python 코드 조각을 입력받아 Gemini 모델에게 설명을 요청하고, 그 결과를 터미널에 출력합니다.
- **주요 기능:**
    - 터미널을 통해 사용자 입력 처리
    - Gemini API(`gemini-1.5-flash` 모델)를 호출하여 코드 설명 생성
    - API 응답을 받아 사용자에게 표시

### `check_model.py`

- **설명:** 사용 가능한 Gemini 모델 목록을 확인하고 특정 모델의 정보를 조회하는 유틸리티 스크립트입니다.
- **작성 배경:**
    - 초기 `app.py` 개발 단계에서, API에 사용할 모델의 정확한 이름(`models/gemini-1.5-flash`)을 확인하고 API 키가 정상적으로 동작하는지 검증할 필요가 있었습니다.
    - 이 스크립트는 `app.py`의 핵심 기능 구현에 앞서, 모델 API의 기본적인 동작을 독립적으로 테스트하고 문제를 진단하기 위한 목적으로 작성되었습니다. 즉, `app.py`를 만들기 위한 사전 검증용 파일입니다.

## 실행 방법

1.  리포지토리를 클론하고 해당 디렉토리로 이동합니다.
2.  필요한 라이브러리를 설치합니다.
    ```bash
    pip install -r requirements.txt
    ```
3.  `.env` 파일을 생성하고 파일 안에 `GOOGLE_API_KEY='실제API키'` 형식으로 자신의 Google API 키를 입력합니다.
4.  터미널에서 `app.py`를 실행합니다.
    ```bash
    python app.py
    ```
